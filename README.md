## 📌 Project Overview

The Vaccination Centre Management System is a web-based platform designed to simplify and automate vaccination appointment management.

---

### 👤 It provides:
- Easy booking for users  
- Efficient workflow for staff  
- Complete control for administrators  

✨ The system reduces manual errors, minimizes waiting time, and improves accessibility.

---

### 🎯 Objectives
- ✔️ Easy online vaccination booking  
- ✔️ Efficient centre & slot management  
- ✔️ Real-time vaccination tracking  
- ✔️ Reduce manual work  
- ✔️ Improve user experience  

---

### 💡 Motivation
- ❌ Traditional systems are slow  
- ❌ No real-time updates  
- ❌ Overcrowding in centres  
- ❌ Poor slot management  

➡️ **Solution:** A smart, automated vaccination management system  

---

### 🚀 Features

#### 👤 User
- Register / Login  
- View centres & slots  
- Book / Cancel appointments  
- Track vaccination status  
- Download certificate  

#### 🏥 Staff
- View daily appointments  
- Verify & check-in users  
- Update vaccination status  

#### 🛠️ Admin
- Manage centres  
- Create slots  
- Manage staff  
- View analytics  

---

### 🌟 Innovative Features

#### 🧠 Smart Slot Recommendation
- Suggests best slot based on:
  - Availability  
  - Crowd load  
  - Timing  

👉 Reduces peak-hour congestion  

#### ⏱️ Waiting Time Estimation
- Displays estimated waiting time
- Based on:
  - Number of users
  - Staff availability
  - Service time

👉 Helps users avoid long queues

---
